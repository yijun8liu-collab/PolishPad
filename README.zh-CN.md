# PolishPad

[English](README.md) | **简体中文**

macOS 菜单栏小工具：全局快捷键唤起悬浮玻璃面板，把口语化、逻辑松散的想法打进去（或语音说进去），回车——AI 重写为结构清晰的文本并**自动粘贴回你来时的应用**。支持多轮对话纠偏、草稿*逐字蜕变*成新文字的等待动画、场景随手切换（优化 / Slack 英文 / 正式 / 精简 / 你自己定义的）。

## 安装（直接下载）

1. 从 [Releases](https://github.com/yijun8liu-collab/PolishPad/releases) 下载最新的 `PolishPad-x.y.z.zip`，解压后把 `PolishPad.app` 拖进「应用程序」
2. 应用没有 Apple 开发者证书签名，首次打开会被 macOS 拦截，两种放行方式：
   - **系统设置 → 隐私与安全性** → 底部点「仍要打开」
   - 或在终端执行 `xattr -cr /Applications/PolishPad.app`
3. 通用二进制（Apple Silicon + Intel）；要求 macOS 13+
4. 按提示授予**辅助功能**权限（自动粘贴回原应用、划词替换需要）

老用户：**设置 → 检查更新 → 下载并安装**——应用自动下载、原地替换并重启到新版（v0.6.1 起；无法原地更新时自动降级为打开下载页）。

## 从源码构建

```sh
./build.sh              # release 构建 + 打包 PolishPad.app
UNIVERSAL=1 ./build.sh  # 通用二进制（arm64 + x86_64）
open PolishPad.app
```

## 首次配置

菜单栏图标 → **设置…** → 填 API 区（任何 OpenAI 兼容端点：DeepSeek / Moonshot / Ollama / 内部代理）：

- **Base URL**：如 `https://api.deepseek.com/v1`
- **API Key**：存储在本机配置文件（`0600` 权限），不使用钥匙串、不会弹密码框
- **模型**：如 `deepseek-chat`

其余项都有合理默认值。配置文件在 `~/Library/Application Support/PolishPad/config.json`，每次请求实时读取——手改立即生效。

## 面板工作流

| 按键 | 动作 |
|---|---|
| `⌃⌥P`* | 唤起 / 收起面板（每次都是全新会话） |
| `↩` | 优化草稿 → 结果流式输出 → **自动粘贴回原应用** |
| 输入 + `↩` | **追加**模式（默认）：新内容优化后并入全文；**修改**模式（`⇥` 切换）：按意见修改——两者都原地替换上一次粘贴 |
| 空 `↩` | 收工关窗（若回退过版本，先把目标替换为当前显示版本） |
| `⌘[` / `⌘]` | 版本切换 · `⌘D` 改动对比 · `⌘N` 重新开始 |
| `Esc` | 停止听写 → 取消请求 → 关窗（分层） |

\* 默认值；设置里点击快捷键字段直接按组合键即可录制（保存后立即生效，录制时按住的修饰键会实时回显）。

等待期间，你的草稿会**逐字飘舞、随流式输出逐字蜕变成新文字**——蜕变速度就是模型的真实生成速度。打开**停顿预取**（设置 → 行为）后，输入停顿时会静默预跑一轮：回车时内容没再改动就瞬间出结果（状态栏带闪电图标；token 消耗说明见设置）。

面板还有：快捷 chips（更短 / 更正式 / 更口语 / 展开）、结果文本就地编辑、语音听写、中/EN 开关（界面 + 输出语言，全应用跟随）、日夜主题切换、右上角设置齿轮。

## 场景

- **内置**：优化（默认）、Slack 英文、正式、精简——提示词全文在设置里**公开可查看、可直接修改**（改了即成为你的覆写版，随时恢复默认）
- **自定义**：可建任意多个、各自命名的场景，中/EN 提示词与名称独立；也可以**用一句话描述让 AI 生成**（面板场景菜单 →「描述创建新场景…」，或设置 →「AI 生成场景」）——多轮协议由本地拼接，生成的场景不会破坏纠偏
- **应用感知**：bundle ID → 场景映射，从 Slack 唤起自动切到 Slack 英文
- **术语表**：`术语=固定译法` 或原样保留，所有场景强制生效

## 划词优化（不开面板）

- `⌃⌥R`：优化当前选中文本并原地替换（菜单栏和右键**服务**菜单里也有）
- `⌃⌥A`：全选 + 优化当前输入框
- 等待期间切换了窗口不会盲贴——结果留在剪贴板并提示手动粘贴
- 被替换的原文会存入**历史**（菜单栏，最近 20 条，重启不丢）；「还原上次替换」可撤销最近一次

处理中光标旁有实时进度气泡；失败时原文绝不被修改，剪贴板内容（含图片）用完自动恢复。

## 隐私

文本只发送到**你自己配置**的端点——指向本地 Ollama（`http://localhost:11434/v1`）或公司内部代理即可完全不出网。API Key 只存在本机（配置文件，`0600` 权限）。无统计上报、无第三方服务。
