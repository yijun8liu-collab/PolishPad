# PolishPad

[English](README.md) | **简体中文**

macOS 菜单栏小工具：全局快捷键唤起悬浮输入框，把口语化、逻辑松散的长段输入交给 AI 重写为结构清晰的文本，结果自动复制到剪贴板；不满意可以多轮对话纠偏。配合自定义提示词，同一套交互也能用于翻译、语气转换等任意文本改写。

## 安装（直接下载）

1. 从 [Releases](https://github.com/yijun8liu-collab/PolishPad/releases) 下载最新的 `PolishPad-x.y.z.zip`，解压后把 `PolishPad.app` 拖入「应用程序」
2. 本应用没有 Apple 开发者签名，首次打开会被系统拦截，任选一种放行：
   - 双击被拒后，去 **系统设置 → 隐私与安全性**，点页面下方的 **「仍要打开」**
   - 或在终端执行：`xattr -cr /Applications/PolishPad.app`
3. 支持 Apple Silicon 和 Intel，需要 macOS 13+

## 从源码构建

```sh
./build.sh        # 编译并打包出 PolishPad.app（需要 Xcode Command Line Tools）
open PolishPad.app
```

开发调试可直接 `swift run`（以终端进程运行，功能相同）。

## 首次配置

启动后点菜单栏的 ✨ 图标 → **打开配置文件**，填写：

```json
{
  "baseURL": "https://api.openai.com/v1",   // 任何 OpenAI-compatible 端点（DeepSeek/Moonshot/Ollama/内部代理均可）
  "apiKey": "sk-...",
  "model": "gpt-4o-mini",
  "temperature": 0.3,
  "maxTokens": 4096,
  "hotkey": "ctrl+option+p",                 // 唤起面板
  "hotkeyPolishSelection": "ctrl+option+r",  // 划词优化替换
  "hotkeyPolishAll": "ctrl+option+a",        // 全选优化替换
                                             // 三个快捷键都可在 设置… 界面直接按键录制，保存即生效
  "promptPreset": "polish",                  // 场景预设：polish / slack-english / formal / concise / custom
  "systemPrompt": null,                      // 预设为 custom 时使用的自定义提示词
  "speechLocale": "zh-CN"                    // 语音识别语言（zh-CN / en-US 等）
}
```

配置文件路径：`~/Library/Application Support/PolishPad/config.json`。
除 `hotkey` 外的字段每次请求时重新读取，改完即生效，无需重启。

## 使用

1. 光标停在任意输入框 → `⌃⌥P` 唤起面板（再按一次收起）
2. 输入原始内容，`Enter` 提交（`Shift+Enter` 换行，中文输入法组字中的回车不会误触发）
3. 优化完成后**自动粘贴回唤起前的应用**，面板保持打开、焦点回到纠偏框
4. 继续输入时用输入框旁的 **追加/修改** 胶囊（`Tab` 键切换）选择语义：**追加（默认）**把你输入的当作新内容，优化后智能并入全文合适位置、已有内容不动；**修改**则是对当前版本提意见（"第二点缩短"）。每轮 `Enter` 后新版本自动删除上一版内容再贴入，**原地替换**；满意 → 留空按 `Enter`、`Esc` 或 ✕ 结束，焦点留在原应用
5. 右上角 **中/EN 开关**：EN 模式下优化结果输出英文（使用原生英文提示词），界面文字同步切换；选择会被记住，划词模式也跟随
6. 请求失败时输入不会丢，可点「复制原文」兜底；每轮结果同时也在剪贴板里
7. **语音输入**：`⌘D` 或点左下角麦克风开始/停止听写，识别文字实时流入当前输入框（组稿框或纠偏框），长停顿不丢内容。首次使用会请求麦克风和语音识别权限；使用 macOS 原生识别，支持本地识别的机器上语音不出网
8. 不想自动粘贴？配置里把 `autoPaste` 设为 `false`，回到"结果只进剪贴板、手动 ⌘V"的模式

每次唤起窗口都是全新会话——关窗即结束上一次对话；窗口开着时可用 `⌘N` 手动清空重来。

## 划词优化（原地替换，不开窗口）

在任意应用里直接优化，全程不离开当前输入框：

- **优化选中文本**：选中文字 → `⌃⌥R` → 原地替换为优化结果
- **全选优化**：光标在输入框内 → `⌃⌥A` → 自动全选、优化、整体替换
- **右键菜单**：选中文字 → 右键 → **服务** → 「PolishPad：优化并替换」/「PolishPad：全选优化并替换」
- **菜单栏入口**：点 ✨ 图标 → 「优化选中文本」/「全选优化替换」（后者不需要先选中任何文字）

处理期间鼠标旁会出现「优化中…」气泡、菜单栏图标变为沙漏，完成后气泡变绿色 ✓ 并播放提示音；失败时弹窗说明，**原文不会被修改**。抓取/回贴临时借用剪贴板，完成后会自动恢复你原来的剪贴板内容（包括图片等非文本）。

注意事项：

- 首次使用会弹出**辅助功能权限**引导（模拟按键需要），在 系统设置 → 隐私与安全性 → 辅助功能 中勾选 PolishPad 后重按快捷键
- 右键「服务」菜单在原生应用（备忘录、Safari、Xcode 等）中可用；VS Code 等 Electron 应用使用自绘右键菜单，不显示服务项，请用快捷键
- 「全选优化并替换」的右键项需要先选中任意文字才会出现（系统服务机制所限）；快捷键无此限制

## 隐私说明

输入内容会发送到你配置的 API 端点。处理敏感内容时，可把 `baseURL` 指向本地 Ollama（`http://localhost:11434/v1`）或公司内网代理，实现零外发。API Key 以明文存于配置文件（v0.2 计划迁移到 Keychain）。
